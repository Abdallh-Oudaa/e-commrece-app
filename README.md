# 🛒 E-Commerce App

## 📱 Description
A complete Flutter-based e-commerce application that helps users explore and purchase products with ease.  
The app is built following clean architecture principles with responsive design and modern state management.

---

## 🚀 Features

- Splash Screen  
- Register Screen  
- Login Screen  
- Home Tab  
- Products Tab  
- Cart Screen  
- Profile Tab  

---

## 🛠️ Tech Stack Used

- Flutter
- REST API
- Shared Preferences
- Clean Architecture
- Repository Pattern
- MVVM Pattern
- Cubit (State Management)
- Responsive UI
- Dependency Injection
